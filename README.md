# Gitus

## Objetivo

Desarrollar un sistema web que integre la API de GitHub para simplificar su uso, haciéndolo más intuitivo y accesible para el entorno educativo.

---

## Problema a resolver

La complejidad de uso de GitHub para usuarios no avanzados, especialmente alumnos, dificultando la gestión y seguimiento de proyectos.

---

## Usuarios

- Alumnos: creación y gestión de proyectos
    
- Profesores: supervisión, descarga y evaluación
    
- Directivos: seguimiento y comentarios
    
- Alcance: toda la escuela
    

---

## Datos y funcionalidades clave

- Integración con API de GitHub
    
- Sistema de roles (alumno, profesor, directivo)
    
- Buscador de usuarios
    
- Comentarios temporales (persistencia de 72h)
    
- Gestión simplificada de repositorios
    

---

## Funcionalidades esenciales

- Vinculación con cuentas de GitHub
    
- Interfaz simplificada para operaciones principales
    
- Navegación intuitiva y accesible
    

---

## Seguridad

- Autenticación (login/registro)
    
- Recuperación de contraseña
    
- Vinculación segura con cuentas de GitHub
    

---

## Requisitos no funcionales

- Alta prioridad en usabilidad
    
- Aplicación web responsive (base para futura app)
    
- Diseño simple, claro y eficiente
    

---

## Diseño

- Identidad visual inspirada en GitHub
    
- Inclusión de logo propio: **Gitus**
    
- Interfaz limpia y enfocada en la experiencia del usuario
